# 01_knapsack_problem

This folder contains a Python implementation of the 0/1 knapsack problem using an evolutionary strategy.

## Files

- `knapsack_problem.py` – implementation of a heuristic knapsack solver that evaluates item selection against a fixed knapsack capacity.
- `knapsack_results.txt` – example output/results generated by the program.

## Description

The program generates a fixed set of item volumes and then applies a simple evolutionary strategy to find a feasible solution that maximizes the packed volume without exceeding capacity.

## How to Run

```bash
python knapsack_problem.py
```
