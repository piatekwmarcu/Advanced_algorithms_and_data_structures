# 02_task_allocation_in_multi-processor_system

This folder contains a Python implementation of a task allocation problem for multiple processors with varying speeds.

## Files

- `task_allocation.py` – implementation of a heuristic evolutionary strategy finding a task-to-processor assignment that minimizes the maximum processor load.
- `output.txt` – example output/results generated by the program.

## Description

The program generates a fixed list of tasks with durations and assigns them to processors of different performance levels. The algorithm attempts to balance the load by minimizing the maximum normalized processing time (`delta_t`).

## How to Run

```bash
python task_allocation.py
```
