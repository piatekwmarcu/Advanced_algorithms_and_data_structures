# 03_asymmetric_traveling_salesman_problem

This folder contains an implementation of the Asymmetric Traveling Salesman Problem (ATSP) and example results.

## Files

- `atsp.py` – Python implementation for solving an ATSP instance using a nearest-neighbor heuristic and 2-opt local improvement.
- `atsp_results.txt` – example summary of initial data and experimental results for several runs.

## Description

The Asymmetric Traveling Salesman Problem is a variant of the Traveling Salesman Problem where travel costs between cities are not symmetric. This folder stores code, cost matrix generation, and results for ATSP experiments.

## How to Run

```bash
python atsp.py
```
